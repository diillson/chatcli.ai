O **ChatCLI** é uma interface de linha de comando que integra os maiores modelos de linguagem (LLMs) diretamente ao seu terminal. Desenvolvido em Go, é rápido, portátil e transforma seu shell em um **assistente de desenvolvimento inteligente** — capaz de entender contexto, executar comandos, editar código e automatizar tarefas complexas.

## Por que usar o ChatCLI?

- **Interface Unificada** — Acesse os melhores modelos do mercado e modelos locais a partir de uma única ferramenta
- **Consciência de Contexto** — Comandos como `@git`, `@file` e `@history` injetam contexto relevante direto no prompt
- **Automação Real** — O Modo Agente transforma a IA em um assistente que executa comandos no seu sistema
- **Engenharia de Código** — O Modo Coder faz a IA ler, editar e testar código em loop com segurança
- **Segurança Integrada** — Comandos perigosos são bloqueados, valores sensíveis são mascarados, policies configuráveis
- **Extensível** — Sistema de plugins, skills, agentes customizados e integração MCP

## Problemas que o ChatCLI resolve

<Info>
- **Copiar e colar interminável** — `cat arquivo.js`, selecionar, Ctrl+C, abrir navegador, colar, digitar pergunta...
- **Mensagens de commit genéricas** — `@git` + uma pergunta = commit message perfeita
- **Análise de logs intimidante** — Pipe direto para a IA: `cat error.log | chatcli -p "causa raiz?"`
- **Curva de aprendizagem** — `@file ./src --mode smart` e pergunte: "como funciona o auth?"
</Info>

## Início rápido

```bash
# Instalar via go install
go install github.com/diillson/chatcli@latest

# Ou baixar binário pronto
# https://github.com/diillson/chatcli/releases
```

```bash
# Configurar provedor
echo 'LLM_PROVIDER=OPENAI' > .env
echo 'OPENAI_API_KEY=sk-xxx' >> .env

# Usar!
chatcli
```
