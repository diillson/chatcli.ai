Bem-vindo à seção de Receitas do **ChatCLI**!

Aqui você encontrará guias práticos, orientados a objetivos, que combinam diferentes funcionalidades do ChatCLI para resolver problemas reais de desenvolvimento e operações. Em vez de focar em um único comando, cada receita mostra um fluxo de trabalho completo.

Use estes guias como ponto de partida para criar suas próprias automações e otimizar suas tarefas diárias.
