---
title: "Delete Runbook"
api: "DELETE /api/v1/runbooks/{name}"
description: "Removes a remediation runbook. Runbooks in use by active incidents cannot be deleted."
---

<ParamField path="name" type="string" required>
  Unique runbook name (e.g., `runbook-disk-cleanup`)
</ParamField>

<ResponseExample>
```json Response 200
{
  "apiVersion": "v1",
  "kind": "Status",
  "metadata": {
    "name": "runbook-disk-cleanup"
  },
  "status": {
    "code": 200,
    "message": "Runbook 'runbook-disk-cleanup' excluido com sucesso",
    "deletedAt": "2026-03-19T17:00:00Z",
    "deletedBy": "admin@empresa.com"
  }
}
```
</ResponseExample>

<ResponseExample>
```json Response 409
{
  "apiVersion": "v1",
  "kind": "Error",
  "error": {
    "code": 409,
    "message": "Runbook in use",
    "details": "The runbook 'runbook-oomkill-standard' is being used by incident INC-20260319-001 and cannot be deleted"
  }
}
```
</ResponseExample>
