---
title: "Approve Request"
api: "POST /api/v1/approvals/{name}/approve"
description: "Approves a pending request, allowing the remediation to be executed"
icon: "circle-check"
---

<ParamField path="name" type="string" required>
  Unique approval name (e.g., `APR-20260319-001`)
</ParamField>

<ParamField body="approvedBy" type="string" required>
  Identifier of the user approving the request
</ParamField>

<ParamField body="comment" type="string">
  Optional comment about the approval
</ParamField>

<ResponseExample>
```json Response 200
{
  "apiVersion": "v1",
  "kind": "Approval",
  "metadata": {
    "name": "APR-20260319-001",
    "namespace": "production"
  },
  "status": {
    "status": "approved",
    "approvedBy": "carlos.silva@empresa.com",
    "approvedAt": "2026-03-19T15:35:00Z",
    "comment": "Aprovado — analise de IA confirma necessidade de aumento de memoria",
    "incident": "INC-20260319-001",
    "remediationTriggered": true
  }
}
```
</ResponseExample>

<ResponseExample>
```json Response 409
{
  "apiVersion": "v1",
  "kind": "Error",
  "error": {
    "code": 409,
    "message": "Approval already decided",
    "details": "The approval 'APR-20260319-001' was already approved by carlos.silva@empresa.com at 2026-03-19T15:35:00Z"
  }
}
```
</ResponseExample>

<ResponseExample>
```json Response 410
{
  "apiVersion": "v1",
  "kind": "Error",
  "error": {
    "code": 410,
    "message": "Approval expired",
    "details": "The approval 'APR-20260319-001' expired at 2026-03-19T16:21:05Z"
  }
}
```
</ResponseExample>
