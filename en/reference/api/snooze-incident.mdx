---
title: "Snooze Incident"
api: "POST /api/v1/incidents/{name}/snooze"
description: "Snoozes notifications for an incident for a specified duration"
icon: "clock"
---

<ParamField path="name" type="string" required>
  Unique incident name (e.g., `INC-20260319-001`)
</ParamField>

<ParamField body="duration" type="string" required>
  Snooze duration in Go duration format (e.g., `30m`, `1h`, `2h30m`)
</ParamField>

<ParamField body="reason" type="string" required>
  Reason for snoozing the incident
</ParamField>

<ParamField body="snoozedBy" type="string" required>
  Identifier of the user snoozing the incident
</ParamField>

<ResponseExample>
```json Response 200
{
  "apiVersion": "v1",
  "kind": "Incident",
  "metadata": {
    "name": "INC-20260319-001",
    "namespace": "production"
  },
  "status": {
    "state": "Analyzing",
    "snoozed": true,
    "snoozedBy": "carlos.silva@empresa.com",
    "snoozedAt": "2026-03-19T15:35:00Z",
    "snoozeUntil": "2026-03-19T16:05:00Z",
    "snoozeReason": "Deploy em andamento, verificar apos conclusao"
  }
}
```
</ResponseExample>

<ResponseExample>
```json Response 400
{
  "apiVersion": "v1",
  "kind": "Error",
  "error": {
    "code": 400,
    "message": "Invalid duration",
    "details": "The 'duration' field must be in Go duration format (e.g., 30m, 1h, 2h30m). Maximum: 24h"
  }
}
```
</ResponseExample>
