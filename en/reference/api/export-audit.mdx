---
title: "Export Audit"
api: "GET /api/v1/audit/export"
description: "Exports the complete audit log in CSV or JSON format for compliance and external analysis"
---

<ParamField query="format" type="string" default="csv">
  Export format: `csv`, `json`
</ParamField>

<ParamField query="since" type="string" required>
  Start date in ISO 8601 format (e.g., `2026-03-01T00:00:00Z`)
</ParamField>

<ParamField query="until" type="string">
  End date in ISO 8601 format (default: now)
</ParamField>

<ParamField query="action" type="string">
  Filter by action type
</ParamField>

<ParamField query="user" type="string">
  Filter by user
</ParamField>

<ParamField query="namespace" type="string">
  Filter by namespace
</ParamField>

The CSV export returns the `Content-Type: text/csv` header and the `Content-Disposition: attachment; filename="audit-export-20260319.csv"` header for automatic download.

For large exports (more than 10,000 records), the API returns a `202 Accepted` with a link for asynchronous download:

<ResponseExample>
```json Response 202 (large export)
{
  "apiVersion": "v1",
  "kind": "AuditExportJob",
  "metadata": {
    "jobId": "export-20260319-001"
  },
  "status": {
    "state": "processing",
    "estimatedRecords": 45000,
    "downloadUrl": "/api/v1/audit/export/download/export-20260319-001",
    "expiresAt": "2026-03-19T22:00:00Z"
  }
}
```
</ResponseExample>
