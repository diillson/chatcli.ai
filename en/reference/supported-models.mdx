  <Tab title="Ollama (Local)">
    Supports any local model via Ollama. Configure in `.env`:

    ```env
    OLLAMA_ENABLED=true
    OLLAMA_MODEL="llama3"
    ```

    Or switch interactively: `/switch --model llama3`

    Use `ollama pull <model>` to download new models.
