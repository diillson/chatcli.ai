---
title: "Reconhecer Incidente"
api: "POST /api/v1/incidents/{name}/acknowledge"
description: "Marca um incidente como reconhecido, indicando que a equipe esta ciente e atuando"
---

<ParamField path="name" type="string" required>
  Nome unico do incidente (ex: `INC-20260319-001`)
</ParamField>

<ParamField body="acknowledgedBy" type="string" required>
  Identificador do usuario ou equipe que esta reconhecendo o incidente
</ParamField>

<ParamField body="message" type="string">
  Mensagem opcional com contexto sobre o reconhecimento
</ParamField>

<ResponseExample>
```json Response 200
{
  "apiVersion": "v1",
  "kind": "Incident",
  "metadata": {
    "name": "INC-20260319-001",
    "namespace": "production"
  },
  "status": {
    "state": "Analyzing",
    "acknowledged": true,
    "acknowledgedBy": "carlos.silva@empresa.com",
    "acknowledgedAt": "2026-03-19T15:30:00Z",
    "message": "Investigando aumento de memoria no payment-service"
  }
}
```
</ResponseExample>

<ResponseExample>
```json Response 409
{
  "apiVersion": "v1",
  "kind": "Error",
  "error": {
    "code": 409,
    "message": "Incidente ja reconhecido",
    "details": "O incidente 'INC-20260319-001' ja foi reconhecido por carlos.silva@empresa.com em 2026-03-19T15:30:00Z"
  }
}
```
</ResponseExample>
