---
title: "Adiar Incidente"
api: "POST /api/v1/incidents/{name}/snooze"
description: "Adia as notificações de um incidente por um periodo especificado"
---

<ParamField path="name" type="string" required>
  Nome único do incidente (ex: `INC-20260319-001`)
</ParamField>

<ParamField body="duration" type="string" required>
  Duracao do adiamento no formato Go duration (ex: `30m`, `1h`, `2h30m`)
</ParamField>

<ParamField body="reason" type="string" required>
  Motivo para adiar o incidente
</ParamField>

<ParamField body="snoozedBy" type="string" required>
  Identificador do usuário que está adiando
</ParamField>

<ResponseExample>
```json Response 200
{
  "apiVersion": "v1",
  "kind": "Incident",
  "metadata": {
    "name": "INC-20260319-001",
    "namespace": "production"
  },
  "status": {
    "state": "Analyzing",
    "snoozed": true,
    "snoozedBy": "carlos.silva@empresa.com",
    "snoozedAt": "2026-03-19T15:35:00Z",
    "snoozeUntil": "2026-03-19T16:05:00Z",
    "snoozeReason": "Deploy em andamento, verificar após conclusao"
  }
}
```
</ResponseExample>

<ResponseExample>
```json Response 400
{
  "apiVersion": "v1",
  "kind": "Error",
  "error": {
    "code": 400,
    "message": "Duracao invalida",
    "details": "O campo 'duration' deve estar no formato Go duration (ex: 30m, 1h, 2h30m). Maximo: 24h"
  }
}
```
</ResponseExample>
