---
title: "Correlações da Federacao"
api: "GET /api/v1/federation/correlations"
description: "Retorna correlações de incidentes entre clusters — incidentes com a mesma causa raiz detectados em multiplos clusters"
---

## Campos de Correlacao

| Campo | Descrição |
|-------|-----------|
| `correlationId` | ID único agrupando incidentes relacionados entre clusters |
| `issue` | Nome do incidente principal |
| `severity` | Severidade do incidente |
| `signalType` | O sinal que disparou a correlacao |
| `correlatedClusters` | Lista separada por virgulas dos nomes dos clusters afetados |
| `elevated` | Se a severidade foi automaticamente elevada devido ao impacto entre clusters |
| `cascade` | Se uma falha em cascata foi detectada (propagacao de staging para producao) |
