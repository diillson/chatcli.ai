---
title: "Aprovar Solicitacao"
api: "POST /api/v1/approvals/{name}/approve"
description: "Aprova uma solicitacao pendente, permitindo que a remediacao seja executada"
---

<ParamField path="name" type="string" required>
  Nome unico da aprovacao (ex: `APR-20260319-001`)
</ParamField>

<ParamField body="approvedBy" type="string" required>
  Identificador do usuario que esta aprovando
</ParamField>

<ParamField body="comment" type="string">
  Comentario opcional sobre a aprovacao
</ParamField>

<ResponseExample>
```json Response 200
{
  "apiVersion": "v1",
  "kind": "Approval",
  "metadata": {
    "name": "APR-20260319-001",
    "namespace": "production"
  },
  "status": {
    "status": "approved",
    "approvedBy": "carlos.silva@empresa.com",
    "approvedAt": "2026-03-19T15:35:00Z",
    "comment": "Aprovado — analise de IA confirma necessidade de aumento de memoria",
    "incident": "INC-20260319-001",
    "remediationTriggered": true
  }
}
```
</ResponseExample>

<ResponseExample>
```json Response 409
{
  "apiVersion": "v1",
  "kind": "Error",
  "error": {
    "code": 409,
    "message": "Aprovacao ja decidida",
    "details": "A aprovacao 'APR-20260319-001' ja foi aprovada por carlos.silva@empresa.com em 2026-03-19T15:35:00Z"
  }
}
```
</ResponseExample>

<ResponseExample>
```json Response 410
{
  "apiVersion": "v1",
  "kind": "Error",
  "error": {
    "code": 410,
    "message": "Aprovacao expirada",
    "details": "A aprovacao 'APR-20260319-001' expirou em 2026-03-19T16:21:05Z"
  }
}
```
</ResponseExample>
