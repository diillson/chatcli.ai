---
title: "Aprovar Solicitação"
api: "POST /api/v1/approvals/{name}/approve"
description: "Aprova uma solicitação pendente, permitindo que a remediação seja executada"
icon: "circle-check"
---

<ParamField path="name" type="string" required>
  Nome único da aprovação (ex: `APR-20260319-001`)
</ParamField>

<ParamField body="approvedBy" type="string" required>
  Identificador do usuário que está aprovando
</ParamField>

<ParamField body="comment" type="string">
  Comentário opcional sobre a aprovação
</ParamField>

<ResponseExample>
```json Response 200
{
  "apiVersion": "v1",
  "kind": "Approval",
  "metadata": {
    "name": "APR-20260319-001",
    "namespace": "production"
  },
  "status": {
    "status": "approved",
    "approvedBy": "carlos.silva@empresa.com",
    "approvedAt": "2026-03-19T15:35:00Z",
    "comment": "Aprovado — análise de IA confirma necessidade de aumento de memória",
    "incident": "INC-20260319-001",
    "remediationTriggered": true
  }
}
```
</ResponseExample>

<ResponseExample>
```json Response 409
{
  "apiVersion": "v1",
  "kind": "Error",
  "error": {
    "code": 409,
    "message": "Aprovação já decidida",
    "details": "A aprovação 'APR-20260319-001' já foi aprovada por carlos.silva@empresa.com em 2026-03-19T15:35:00Z"
  }
}
```
</ResponseExample>

<ResponseExample>
```json Response 410
{
  "apiVersion": "v1",
  "kind": "Error",
  "error": {
    "code": 410,
    "message": "Aprovação expirada",
    "details": "A aprovação 'APR-20260319-001' expirou em 2026-03-19T16:21:05Z"
  }
}
```
</ResponseExample>
