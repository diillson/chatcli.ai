---
title: "Obter Cluster"
api: "GET /api/v1/clusters/{name}"
description: "Retorna detalhes de um cluster federado especifico, incluindo status de saude, contagem de nos, uso de recursos e incidentes ativos"
---

<ParamField path="name" type="string" required>
  Nome unico de registro do cluster
</ParamField>

<ParamField query="namespace" type="string" default="default">
  Namespace do Kubernetes onde o CR ClusterRegistration reside
</ParamField>

<ResponseExample>

    "code": 404,
