---
title: "Excluir Runbook"
api: "DELETE /api/v1/runbooks/{name}"
description: "Remove um runbook de remediacao. Runbooks em uso por incidentes ativos nao podem ser excluidos."
---

<ParamField path="name" type="string" required>
  Nome unico do runbook (ex: `runbook-disk-cleanup`)
</ParamField>

<ResponseExample>
```json Response 200
{
  "apiVersion": "v1",
  "kind": "Status",
  "metadata": {
    "name": "runbook-disk-cleanup"
  },
  "status": {
    "code": 200,
    "message": "Runbook 'runbook-disk-cleanup' excluido com sucesso",
    "deletedAt": "2026-03-19T17:00:00Z",
    "deletedBy": "admin@empresa.com"
  }
}
```
</ResponseExample>

<ResponseExample>
```json Response 409
{
  "apiVersion": "v1",
  "kind": "Error",
  "error": {
    "code": 409,
    "message": "Runbook em uso",
    "details": "O runbook 'runbook-oomkill-standard' esta sendo usado pelo incidente INC-20260319-001 e nao pode ser excluido"
  }
}
```
</ResponseExample>
