## AutenticaĂ§ĂŁo OAuth

AlĂ©m das chaves de API tradicionais, o ChatCLI suporta **autenticaĂ§ĂŁo via OAuth** para OpenAI, Anthropic e GitHub Copilot. Com o OAuth, vocĂȘ pode usar seu plano existente (ChatGPT Plus, Codex, Claude Pro, GitHub Copilot) sem gerar API keys.

| VariĂĄvel | DescriĂ§ĂŁo | PadrĂŁo |
| :--- | :--- | :--- |
| `CHATCLI_AUTH_DIR` | DiretĂłrio onde as credenciais OAuth sĂŁo armazenadas. | `~/.chatcli/` |
| `CHATCLI_OPENAI_CLIENT_ID` | Permite sobrescrever o client ID do OAuth da OpenAI. | (interno) |

> Use `/auth login openai-codex`, `/auth login anthropic` ou `/auth login github-copilot` no modo interativo para iniciar o fluxo OAuth. Consulte a [documentaĂ§ĂŁo completa de OAuth](/features/oauth-authentication) para mais detalhes.
