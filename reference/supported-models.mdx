  <Tab title="Ollama (Local)">
    Suporta qualquer modelo local via Ollama. Configure no `.env`:

    ```env
    OLLAMA_ENABLED=true
    OLLAMA_MODEL="llama3"
    ```

    Ou troque interativamente: `/switch --model llama3`

    Use `ollama pull <modelo>` para baixar novos modelos.
